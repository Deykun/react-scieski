# scieski
W założeniu skrypt ma na podstawie tras Endomondo generować mapy których domyślnie serwis ten nie generuje.

## Działa:
- Import tras z plików .tcx i .gpx
- Generowanie mapy
- Tłumaczenia: en, pl
